Authority: `folke/tokyonight.nvim` release `v4.14.1`, commit
`545d72cde6400835d895160ecb5853874fd5156d`.

Night, Storm, Moon, and Day use the exact colors tables in the four pinned extras. They use the flat
Simplified schema so secondary syntax, diagnostic, markup, version-control, and UI roles follow
Neotheme's fixed checked fan-out. Repeated assignments are deliberate best-fit aliases, not claims
that upstream assigned all derived Neotheme roles. No value is interpolated or synthesized.

The pinned license is Apache-2.0. It contains the standard appendix placeholder but no
project-specific copyright notice, so metadata explicitly records `upstream-not-provided` and an
empty notice rather than inventing a holder.
