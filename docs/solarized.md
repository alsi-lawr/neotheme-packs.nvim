# Solarized mapping

Authority: `altercation/vim-colors-solarized` at
`528a59f26d12278698bb946f8fb82a63711eec21`.

Solarized is intentionally a sixteen-color system, so both variants use Neotheme's flat Simplified
schema. The light record follows the pinned source's base03/base3, base02/base2, base01/base1, and
base00/base0 swaps. Neotheme's broader diagnostic, markup, version-control, syntax, and UI graph is
therefore derived by the checked core transform. Repeated values are deliberate best-fit aliases,
not claims that upstream assigned all Neotheme roles. All hexadecimal values are exact pinned GUI
constants; none are synthesized.
