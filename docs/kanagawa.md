# Kanagawa mapping

Authority: `rebelot/kanagawa.nvim` at `bb85e4bfc8d89b0e62c8fa53ccdd13d12e2f77b3`.
The Wave, Dragon, and Lotus records retain upstream hexadecimal tokens in Full mode.
Kanagawa exposes more semantic groups than Neotheme and the two models do not align exactly;
markup headings, link labels, unchecked items, search variants, and some syntax roles therefore use
the nearest upstream semantic token. These are explicit best-fit aliases, not claims that upstream
assigned those Neotheme roles. No color values are interpolated or synthesized.
